# rime_latex
Difficulty typing mathematical symbols?

Try Rime-latex for Latex-based Symbols:)


# RIME_LaTeX's extension to tex symbols

## Number superscript/subscript
`\o1\u1` (over\under) to type `¹₁`, 
```
\u1 ->
```
